# Scoop Submission Readiness

This folder contains maintainer-facing submission checklists for contracts whose Scoop target is implemented.

Submission rule:
- The manifest must install from a public URL.
- The install must avoid elevation.
- The package must have hardening and no-admin smoke evidence reports.
- The PR must not claim GUI launch or uninstall cleanup beyond evidence actually collected.

Current implemented manifests:
- `nomachine-enterprise-client`
- `omnivoice-studio`
- `pinokio`
- `tightvnc-viewer`
- `voicebox`
