# Track Spec: Installed-App Package Candidate Triage

## Goal
Keep installed-app discoveries in a governed triage lane so they can become Scoop/Chocolatey package candidates only after duplicate lookup, no-admin feasibility, and source provenance are established.

## Scope
- DAX Studio
- DoublShot
- GitHub Copilot Desktop
- Linear
- Mixxx
- Power Platform ToolBox
- XrmToolBox

## Out of Scope
- Publishing Chocolatey packages from this repo.
- Treating installed files as package sources.
- Marking any candidate hardened-complete without runtime smoke evidence.
