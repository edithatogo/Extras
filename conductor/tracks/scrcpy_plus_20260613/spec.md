# Track Spec: SCRCPY+ No-Admin Packaging

## Goal
Determine whether SCRCPY+ can be part of the no-admin package factory as implemented, deferred, or blocked.

## Scope
- Contract: `packaging/contracts/scrcpy-plus.json`
- Related sources: `Frontesque.scrcpy+`, release artifacts, article-defined flow.

## Acceptance Criteria
- Decision is explicit and evidence-backed (installed behavior, duplicate lookups, and manifest availability).
- Non-admin strategy is not claimed without reproducible test path.
